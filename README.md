# SteamJSTracker

## A JavaScript based twitter bot to keep track of any changes done to a Steam Application.
Above work is archived from an old project.

### Build Requirements : 
***NPM Packages***

`-> puppeteer + stealth plugins`

`-> Twitter-API-V2`

`-> html_compare`

`-> json-diff`

`-> steam-user`

`-> axios`

`-> pastebin_api` *optional*
